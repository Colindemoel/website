# 3.0 Verkenning

> ## "Steal like an artist"

Nu mijn analyse fase is afgerond, is het tijd om eens goed naar de concurrenten te gaan kijken. Hoe is hun informatie architectuur? Hoe lossen zij bepaalde problemen op? Welke design patterns komen er vaak terug? Waarom zou ik namelijk zelf helemaal het wiel opnieuw uitvinden terwijl dit niet nodig is.  

Ook heb ik een inspiratie bord gemaakt in illustrator waarop ik veel verschillende interessante en mogelijke oplossingen heb verzameld. Hierdoor kan ik snel mijn gevonden inspiratie weer terug vinden.
